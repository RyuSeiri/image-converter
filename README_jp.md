## ✔ このプロジェクトについて

画像を処理するために Python を使用するプロジェクトです

主に使用する package は次のとおりです。

- Flask

- Pillow

具体的な実行手順は次のとおりです。

1.まず次のコマンドを実行します

```sh

$ pip install -r requirements.txt

```

2.次に、`start.bat`をダブルクリックして python プログラムを実行します

3.最後にブラウザ入力コンソールの表示出力の URL を開いてサイトにアクセスする
