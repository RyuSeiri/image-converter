## ✔ 关于这个工程

这是一个使用 Python 用于处理图像的工程
主要用到的 package 如下：

- Flask
- Pillow

具体的执行步骤如下：

1. 首先执行下面的命令

```sh
$ pip install -r requirements.txt
```

2. 然后双击`start.bat`运行 python 程序

3. 最后打开浏览器输入控制台显示输出的 URL 访问该网站
